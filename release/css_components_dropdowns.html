Alignment

By default, a dropdown menu is automatically positioned 100% from the top and along the left side of its parent. Add .dropdown-menu-right to a .dropdown-menu to right align the dropdown menu

Dropdown-Driven Tabs

Allows a dropdown component to control tabbed content in a separate <div>, tying each dropdown option link to a corresponding tab by id.

Use the dropdown-driven-tabs class on the dropdown to attach a click event listener that powers the tab activation.

Add the active class to the default tab's dropdown <li> so it is selected on initial state.

Mix and match standard dropdown and tab styling as desired.

  • Spaces
  • Domains
  • Members

Domains Content

  <button id="dropdown-button-1" class="dropdown-toggle btn btn-default em-low" type="button" data-toggle="dropdown" aria-haspopup="true" aria-expanded="false">
    Button Dropdown
    <svg class="icon caret"><use xlink:href="#i-arrow-down-selected"/></svg>
  </button>
